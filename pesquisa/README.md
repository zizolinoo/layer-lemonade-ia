# Pesquisa

Area para preparar pautas, registrar achados e organizar checagem.

## Uso sugerido

- abrir uma pauta em `template-pauta.md`
- registrar verificacao em `template-checagem.md`
- criar arquivos novos a partir dos templates conforme o projeto crescer

## Regra de ouro

Informacao util aqui deve ter contexto, data e fonte.
