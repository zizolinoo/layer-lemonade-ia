# Layer Lemonade IA

Repositorio base para documentacao, contexto e operacao da Layer Lemonade IA.

## Objetivo

Centralizar em um unico lugar:

- documentacao do projeto
- contexto de persona e direcionamento criativo
- rotinas operacionais
- referencia para uso local com espelhamento no GitHub

## Estrutura inicial

- `docs/`
  Documentacao viva do projeto, contexto, referencias e definicoes.
- `operations/`
  Rotinas de trabalho, checklists e processos recorrentes.

## Fluxo de uso

2. Versionar alteracoes com Git
3. Usar o GitHub como espelho remoto e backup operacional
