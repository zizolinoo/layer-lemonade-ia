# Layer Lemonade IA

Sistema pessoal de apoio para pesquisa, organizacao editorial, checagem e construcao de roteiros para os conteudos que voce vai produzir para o Layer Lemonade.

## O que este repositorio e

- um cerebro operacional para ajudar na producao de conteudo
- uma base para pesquisa e atualizacao sobre video com IA
- um apoio para ideias, angulos, verificacao e estrutura de roteiro
- um espelho local + GitHub para acessar o projeto em outras maquinas

## O que este repositorio nao e

- nao e a ferramenta final de geracao dos videos
- nao e um arquivo morto generico
- nao e um repositório para guardar qualquer assunto fora do contexto de video com IA

## Estrutura

- `docs/`
  Contexto do canal, sistema, persona, linha editorial e referencias-base.
- `pesquisa/`
  Templates e materiais para pautas, fact-check e monitoramento.
- `roteiros/`
  Estruturas para construir roteiros e enredos.
- `operations/`
  Fluxo de trabalho, checklists e rotina de atualizacao.
- `handoff/`
  Arquivos para abrir o contexto atual em outro Codex.

## Fluxo sugerido

1. Definir a pauta ou a pergunta de pesquisa
2. Investigar ferramentas, movimentos do mercado e referencias
3. Checar fatos e registrar fontes
4. Encontrar o melhor angulo editorial para o Layer Lemonade
5. Estruturar o roteiro
6. Sincronizar o aprendizado e o material no GitHub

## Contexto central

- Projeto: `Layer Lemonade IA`
- Uso: pessoal e profissional
- Funcao: apoiar sua producao de conteudo para o Layer Lemonade
- Foco tematico: video com IA, ferramentas, fluxos, modelos e implicacoes criativas
- Saida principal: pesquisa organizada e roteiros melhores
