# News Prints 2026-04-22

Aqui `print` significa captura vertical de noticia, pagina oficial ou tela de referencia, pensada para entrar no Reels como scroll, crop ou pan.

## Regra visual

- largura base: `1080`
- altura: `1920` ou maior
- proporcao: linguagem de celular
- uso: scroll, zoom-in, crop em titulo, corte para destaque de trecho

## Tipos de print nesta pasta

- prints diretos de paginas oficiais
- prints rolaveis em full page
- prints browser-like montados a partir de noticias reais quando a pagina original nao responde bem em headless

## Observacao

Estes assets foram pensados para montagem rapida no Premiere, nao para design final.
