# Checklist de Atualizacao

## Antes de pesquisar

- confirmar a pergunta central
- checar se o tema conversa com video, IA e motion
- abrir uma pauta antes de sair coletando link sem criterio

## Durante a pesquisa

- registrar data e fonte
- separar fato, interpretacao e palpite
- evitar repetir hype sem verificar

## Antes de escrever roteiro

- validar se o angulo esta claro
- checar se ha pelo menos uma utilidade pratica para o publico
- confirmar se os pontos principais estao sustentados

## Fechamento

- atualizar a documentacao necessaria
- commitar com mensagem clara
- sincronizar com o GitHub
- deixar o handoff pronto se for continuar em outra maquina ou outro Codex
