# Premiere Reels Template

Projeto-base de Premiere para os 3 Reels quentes de IA visual em formato vertical.

Arquivos principais:
- `Layer-Lemonade-IA-Reels.prproj`: projeto com as 3 sequencias criadas.
- `scripts/setup-3-sequencias-reels.jsx`: script para recriar a estrutura no Premiere Beta.

Sequencias:
- `SEQ_2026-04-22_GPT_IMAGES_2_0`
- `SEQ_2026-04-22_DEEPFAKE_SEGURANCA`
- `SEQ_2026-04-22_BENCHMARK_MJ_GPT_CLAUDE`

O que ja esta pronto:
- projeto vertical 9:16 a 30 fps
- prints importados
- marcadores com a ordem da timeline
- pontos de insercao dos prints em V3

O que ainda e manual:
- inserir o video principal gravado no iPhone
- substituir os placeholders do benchmark pelas imagens finais geradas
- ajustar mix final de audio, cor e legendas conforme a gravacao

Para recriar do zero no Premiere Beta:
1. Abrir o Premiere Beta.
2. Executar `scripts/setup-3-sequencias-reels.jsx`.
3. Conferir as 3 sequencias e os marcadores.
